Origin

Academic project

LyricsGuesser is developed as an academic project by students of Università degli Studi di Milano.

Previous knowledge

We used our knowledge of the Spotify API matured in other studies as a starting point in order to offer an engaging website.

Goals

Deliver an entertaining platform revolving around songs the user is familiar with.

Focus of the website is a simple but effective interface with some tweaks for mobile users.

Development

Technologies used

The website makes use of the HTML5 APIs to store client-side data with LocalStorage.

The styling is made entirely with CSS stylesheets and divided per page content. Some interesting tricks used are the home page gradient text and the parallax effect. We also used FontAwesome for the page icons.

There's a backend server with over 1.1k lines of Javascript developed with NodeJS and Express. We use this server to store user and game data (sensible data is encrypted) and to communicate with external APIs (such as Spotify's and Musixmatch's).

The website itself runs on a Debian machine on an Apache web server.

Issues faced

Since songs lyrics are protected by Copyright, a website is not allowed to show them without proper permissions or privileges.

We use the Musixmatch API in order to retrieve those lyrics but we use them solely with academical purposes and with no commercial intent.

All the Copyright info are shown with the corresponding lyrics.

The Spotify logo is always used following the instructions reported on the official page and we do not take credit for it.
